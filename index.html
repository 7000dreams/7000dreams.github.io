Welcome!

This website is a personal website which is a repository of some of my writings, reflections and activities. The Academics page will contain links to my academic writing and other academic materials.

The ClarityOne page is a side-project: this contains downloadable links to some Mac Apps that I have created. The basic version of the apps can be downloaded from the ClarityOne page. Zip files can be downloaded, and on extraction, a .app executable application will be available. ClarityOne is the name of the apps; currently, I have tge preliminary versions which are still works in progress. The goal is ultimately to have working versions of Clarity Explorer (a file explorer), Clarity Editor (a rich text editor), Clarity Calendar (a minimalist calendar), Clarity Clip (a clipboard manager) and ClarityWiki (a personal knowledge management and note-taking system)

The Mountains link takes you to a distinct section of this website. This section contains my travel-related writings and photographs. Most of the entries are based on my mountain journeys: treks and climbs, wildlife safaris in the foothills of the mountains, or just leisure travel in the hills.
